# Analisis Data K-pop Idol 🎤
Proyek ini menganalisis karakteristik fisik dan tren industri K-pop berdasarkan data publik.

### 📊 Hasil Temuan Utama:
- **Distribusi Gender:** Seimbang antara idol pria dan wanita.
- **Fisik:** Rata-rata tinggi badan pria ~177cm dan wanita ~164cm.
- **Asal Negara:** Didominasi Korea Selatan, diikuti Jepang dan China.

### 🛠️ Teknologi:
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab / Jupyter Notebook
